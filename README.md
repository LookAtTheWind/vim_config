# vim_config

### 安装
+ mkdir ~/vim_config
+ cd ~/vim_config
+ copy the vimrc, basic.vim, maps.vim, plugs.vim, plugs_config.vim, markdown.vim to this folder
+ create ~/vim/syntax, then copy the c.vim to this folder
### 说明
##### vimrc
+ 将你的~/.vimrc备份，替换
##### basic.vim
+	基本配置
##### maps.vim
+	个人的一些映射
##### plugs.vim
+	vim插件
##### plugs_config.vim
+	vim插件自定义设置
##### markdown.vim
+	markdown自定义一些映射
##### c.vim
+ 函数高亮
---
> 至于有的插件安装会报错的情况，希望你自己百度解决，这样你以后安装别的插件也会有经验。
